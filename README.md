# go-tcp-chat

## Simple tcp chat

## Installation

> `git clone git@github.com:teimurjan/go-tcp-chat.git`

> `cd go-tcp-chat`

> `export GOPATH=$(pwd)`

> `make`

To run server

> `srv`

To run client

> `cli`

To remove all object files

> `make clean`

To compile sources and run server

> `make run`
